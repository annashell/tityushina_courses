Введение в React

React — это JavaScript-библиотека для создания пользовательских интерфейсов.

Основные концепции

  • Компонентный подход
  • Виртуальный DOM
  • Однонаправленный поток данных
Примечание: React создан Facebook и используется в таких приложениях как Instagram и WhatsApp.

Пример кода

function HelloMessage({ name }) {
    return <div>Привет, {name}!</div>;
}
